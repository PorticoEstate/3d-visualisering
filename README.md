# 3D Visualisering av sammenhenger

Et interaktivt 3D visualiseringsverktøy for å utforske sammenhenger i datasett.

## Beskrivelse

Dette prosjektet tilbyr en 3D grafisk representasjon av data hvor brukere kan:

- Navigere i 3D-rommet ved å klikke og dra
- Zoome inn og ut med rullehjulet
- Utforske sammenhenger mellom datapunkter

## Filer

- `index.html` - Hovedfilen som inneholder HTML, CSS og JavaScript
- `data/` - Mappe som inneholder datafilene:
  - `datasets.json` - Datasett informasjon
  - `ecosystem.json` - Økosystem data
  - `masterdatabase.json` - Hoveddatabase

## Hvordan bruke

1. Åpne `index.html` i en moderne nettleser
2. Bruk musen til å navigere i 3D-rommet:
   - Klikk og dra for å rotere visningen
   - Bruk rullehjulet for å zoome inn/ut

## Teknologi

- HTML5
- CSS3
- JavaScript
- 3D grafikk (Three.js eller lignende)
